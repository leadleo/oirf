# Engineering dimension database

Use this as the internal dimension database. Do not expose dimension codes in external-facing frameworks.

## 1. 产业定义与历程

> 通过界定产业物理边界、复盘历史关键拐点及评估全球分工位势，确认行业在经济系统中的底层坐标与生命周期阶段。

- 1.1 产业内涵与统计标准
- 1.2 细分赛道与产品分类
- 1.3 发展历程与关键里程碑
- 1.4 产业生命周期与特征
- 1.5 资源要素禀赋
- 1.6 产业标准与协议
- 1.7 全球产业分工与国际对标

## 2. 宏观环境与规制

> 整合宏观/产能周期波动、政策导向、制度壁垒及ESG软约束，界定产业运行的合法性边界与资源承载力极限。

- 2.1 宏观与产业周期
- 2.2 产业扶持与导向政策
- 2.3 监管准入与制度性壁垒
- 2.4 全球贸易与地缘政治风险
- 2.5 政策传导与执行效能
- 2.6 ESG与可持续治理
- 2.7 国际政策路径对比

## 3. 产业链结构与价值链

> 通过绘制上下游拓扑结构，解析成本结构与利润空间的非对称分布，识别产业链内权力重心的迁移路径与议价权底数。

- 3.1 产业链结构梳理
- 3.2 价值链利润分布
- 3.3 行业核心成本构成要素
- 3.4 议价权与利润迁移

## 4. 市场空间与竞争格局

> 化分析横向市场集中度与准入门槛，评估技术/品牌壁垒的深度，并测算生态协作网络对竞争优势的非线性杠杆作用。

- 4.1 市场集中度与竞争格局
- 4.2 品牌、技术与转换成本
- 4.3 市场规模建模预测
- 4.4 行业进入壁垒与退出障碍
- 4.5 产业生态与协同网络

## 5. 技术、产品与定价

> 评估技术路线演进与科研储备，重点考察组织执行力将研发投入转化为差异化产品与超额定价权的损耗与效能。

- 5.1 技术路线博弈演进
- 5.2 人才储备与科研基础
- 5.3 研发投入效率与专利转化

## 6. 用户、场景与决策
